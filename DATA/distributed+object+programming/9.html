About the Job

Bishop & Co. is seeking qualified candidates for 2 direct hire .NET Developer openings on Maui (Kihei). Qualified candidates must possess or be eligible for a federal clearance.

Core Responsibilities:

1. Develop Web and desktop based applications/services following design specifications provided by Architects

2. Develop graphical user interfaces with a high level of human factors experience

3. Participate in standard code cycle releases to development environment

4. Support production deployment of software to ensure successful transition to operations and maintenance

5. Design test cases around code modules to evaluate functionality

6. Technical documentation, code refactoring, performance analysis

7. Participate in daily meetings and other Agile processes to ensure quality rapid capability delivery


Minimum Qualifications:

  1. 3+ years of experience developing applications with .NET platforms (C#, VB.NET, ASP.NET, etc.)
  2. 3+ years of experience with database development with respect to supporting .NET based applications (SQL Server, Oracle, MySQL, etc.)
  3. Silverlight/WPF, Web Services, DB Programming
  4. HTML, JavaScript, DHTML, XML/XSL, CSS
  5. Object oriented design/development
  6. Experience working in a distributed/source control/configuration control environment
  7. Must possess an active clearance or be eligible for a federal clearance


Additional Qualifications:

  1. Experience developing services-based software endpoints that can be utilized within a Services Oriented Architecture (SOA)
  2. Experience with Entity Framework, Flex, Flash, ActionScript
  3. Hands on knowledge of design patterns like MVVM, MVC, MVP, Composite, Builder, Factory, Template, Strategy, Command, etc.


Preferred Qualifications:

1. Knowledge of WCF, Windows Workflow Foundation, Windows Forms, Windows Azure, NIEM

2. Knowledge of Geospatial Information System (GIS) design and/or implementation

3. Experience working in an Agile environment

For immediate consideration, please forward your resume to jobs@bishopco.net.
