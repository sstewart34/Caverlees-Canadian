About the Job


Systems Specialist

Position Description

 

Summary

Ready to join an industry leader? ArcaSearch preserves heritage and creates access to knowledge through the digitization of documents, books or records. Using a customized scanning/capture system, ArcaSearch created patented technology and capability of scanning single documents to bound volumes including materials like paper, newspaper, onion skin, hardboard, carbon paper, blue prints and photographic paper all while maintaining accuracy, clarity and color.  The document delivery system uses a cloud based application that may run on the Internet, Intranet, or a private network.

As a Systems Specialist, you will play a central role in the digital processing of digitized documents, large scale file management, and managing our cloud based application archival interface. In addition, you will help develop highly complex internet and intranet applications on one or more diverse platforms. This position will be expected to provide a keen eye on process improvement. In addition, a Systems Specialist will also interact directly with our Document Imaging Technician team, which facilitates the image capture portion of the ArcaSearch service.

Position Overview

Directly responsible for organizing and managing large amount of files created from digitization process. Projects at times will span multiple work groups. Once organized, the Systems Specialist will utilize furnished ArcaSearch customized scripts to process digitized documents into final deliverable formats. After quality control review, deliverables will be written to external media, backups created, verified, and external media used during production is to be cleared.

Uses advanced knowledge of web technologies, database modeling tools and/or database interactions. Evaluates systems specifications for client area web site data management requirements and determines and implements the most efficient and cost-effective software/package solution. Researches and tracks new web technologies, develop service applications, and analyze business requirements as they pertain to web-based data management systems. Responsible for system quality assurance and consistency with installation and security policy and standards.

Core Responsibilities

         Manage multiple applications or tools on one or more platforms

         Acts as an intermediary for tech related production troubleshooting guidance

         Manage data process for applications, tools, website and portal monitoring

         Responsible for system quality assurance and consistency with installation and security policy

         Uses knowledge of applications (web and non-web based) and tool technologies to facilitate or participate in projects resulting in functional enhancements of applications

         Evaluates each process for final product based on client requirements

         Uses programming tools to write scripts as needed or required

         Performs web-enabled database development and module design

         Performs application security activities and participates in security planning

         Performs duties within security requirements and audits
